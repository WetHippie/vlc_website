The standard Java image loaders that come with the JDK are terrible. The ones from the NIO package aren't much better. They are quite slow and consume extremely large amounts of memory. A problem since JDK 1.1, and still an issue with Java 8, these libraries plug into the bottom of the image loading framework and use the native libraries provided by the platform to improve speed and memory usage.
